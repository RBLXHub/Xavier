    2.  Click Add Tunnel

    5.  Select **TCP+UDP (protocol)** in tunnel type
    6.  Set **Local Port** to `53640` and click Add tunnel

1.  **Identify the details:**

    * **port:** Port, yeah

2.  **Configure the Launcher:**
